# Git Worktree Visual Manager Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a macOS-first Tauri MVP that scans real local Git repositories, visualizes repos/worktrees/branches on a canvas, and executes key Git operations through safety previews.

**Architecture:** The frontend is a React/TypeScript app using React Flow for the canvas and an inspector panel for actions. The backend is a Rust Tauri command layer that owns repo discovery, Git command execution, SQLite persistence, and safety previews. Git behavior is verified with generated sandbox repositories so destructive cases are repeatable.

**Tech Stack:** Tauri v2, React, TypeScript, Vite, React Flow, Rust, SQLite, system `git`, Node test tooling, Rust unit tests.
